CHAT COMMANDS HELP

Basic Commands:

  • /r - Refresh messages
  • /next - Next page
  • /prev - Previous page
  • /auto - Toggle auto-refresh
  • /clear - Clear screen
  • /status - Show connection status
  • /exit - Quit chat
  • /help - Show this help

Message Format:

  • Other users - Messages from others
  • You - Your messages
  • System - System notifications

Admin/Owner Commands:

  • /ban [reason] - Ban a user
  • /unban - Unban a user
  • /check - Check user status
